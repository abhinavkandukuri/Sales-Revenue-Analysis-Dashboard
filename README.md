# 📊 Sales & Revenue Analysis Dashboard

An interactive and visually engaging **Power BI Dashboard** built to analyze sales performance, revenue growth, customer trends, and business profitability. This project transforms raw sales data into meaningful insights using dynamic charts, KPI cards, slicers, and business intelligence techniques.

## 🚀 Project Highlights

✨ Interactive dashboard with modern UI design
✨ Real-time business insights using visual analytics
✨ KPI tracking for Sales, Profit, Quantity & AOV
✨ Category-wise and State-wise performance analysis
✨ Customer and Payment Mode insights
✨ Monthly profit trend visualization for decision-making

---

## 📌 Key Features

### 📈 KPI Metrics

* Total Sales Amount
* Total Profit
* Total Quantity Sold
* Average Order Value (AOV)

### 🌍 Sales Analysis

* State-wise Sales Performance
* Category & Sub-Category Analysis
* Customer Purchase Comparison

### 💳 Payment Insights

* Payment Mode Distribution
* COD, UPI, Debit Card & Credit Card Analysis

### 📅 Trend Analysis

* Monthly Profit Tracking
* Profit Growth & Loss Visualization

### 🎛️ Interactive Elements

* Quarter Filters
* Dynamic Slicers
* Responsive Visual Reports
